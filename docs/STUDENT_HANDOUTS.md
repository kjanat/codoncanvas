**Part B:**

1. **Missense** - GGA (CIRCLE) â†’ CCA (RECT), shape changes from circle to rectangle

## Handout 3: Base-4 Number Encoding Guide

### What is Base-4?

Normal numbers (base-10) use digits 0-9.
Base-4 uses only **4 digits: 0, 1, 2, 3**

In CodonCanvas:

- **A = 0**
- **C = 1**
- **G = 2**
- **T = 3**

### How to Calculate

**Formula:** `value = d1 Ã— 16 + d2 Ã— 4 + d3`

**Example: CCC**

- C = 1, C = 1, C = 1
- 1Ã—16 + 1Ã—4 + 1 = 16 + 4 + 1 = **21**
